# Обновление: Страница Анти-чит

## 🎮 Что добавлено

### Навигация
- **3 кнопки в хедере**: Анти-чит (слева), Главная (центр), Проверка (справа)
- Активная кнопка подсвечивается градиентом
- Плавное переключение между страницами

### Страница "Анти-чит"
- Отображает всех игроков онлайн на серверах Fear Protection
- **Новый дизайн карточек** (как на скриншоте):
  - Аватар игрока (80x80px, круглый)
  - Никнейм и Steam ID
  - Индикатор возраста аккаунта (красный = новый, зеленый = старый)
  - Информация о сервере с IP
  - Статистика: Убийства, Смерти, Пинг
  - Кнопки STEAM и FEAR (розовая обводка)
  - Кнопки копирования STEAMID и IP
  - Большая кнопка ПОДКЛЮЧИТЬСЯ (градиент розовый-фиолетовый)

### Функционал
- ✅ Автоматическая загрузка игроков при открытии страницы
- ✅ Автообновление каждые 30 секунд
- ✅ Подключение к yooma.su WebSocket при загрузке
- ✅ Копирование Steam ID и IP в буфер обмена
- ✅ Прямое подключение к серверу через Steam
- ✅ Ссылки на профили Steam и Fear

## 📁 Новые файлы

1. **js/AntiCheatPage.js** - класс для страницы Анти-чит
2. **index.html** - обновлен с навигацией и тремя страницами

## 🎨 Дизайн

### Цветовая схема
- Фон карточек: `rgba(40, 20, 20, 0.6)`
- Обводка: `rgba(255, 100, 100, 0.3)`
- Кнопки: розовый `#ff1493` и фиолетовый `#9c27b0`
- Градиент кнопки подключения: `linear-gradient(90deg, #ff1493 0%, #9c27b0 100%)`

### Индикаторы
- 🔴 Красный круг = аккаунт младше 30 дней
- 🟢 Зеленый круг = аккаунт старше 30 дней

## 🚀 Запуск

```bash
cd check-page-standalone
node server.js          # Порт 3002
node yooma-proxy.js     # Порт 3003 (WebSocket)
```

Откройте: http://localhost:3002

## 📊 API

Страница использует:
- `/api/fear/servers/` - список серверов
- Steam API - информация об игроках
- WebSocket `ws://localhost:3003` - проверка yooma.su банов

## ⚙️ Настройки

- **Интервал обновления**: 30 секунд (можно изменить в `AntiCheatPage.js`)
- **Порог "нового аккаунта"**: 30 дней (можно изменить в методе `calculateAccountAge`)

## 🔧 Технические детали

### Структура страниц
```
index.html
├── Анти-чит (anticheat-page)
│   └── Список игроков онлайн
├── Главная (home-page)
│   └── Приветственный экран
└── Проверка (check-page)
    ├── Проверка Fear
    └── Проверка Fear + yooma.su
```

### Классы JavaScript
- `AntiCheatPage` - управление страницей анти-чит
- `ConfigChecker` - проверка config.vdf (Fear)
- `ConfigCheckerUMA` - проверка config.vdf (Fear + yooma.su)
- `APIClient` - работа с API

## 📝 Примечания

- Репорты **НЕ** отображаются на странице Анти-чит (только игроки онлайн)
- Дизайн карточек соответствует предоставленному скриншоту
- Все данные обновляются автоматически
