# Changelog - Check Page Standalone

## Последние изменения (13.05.2026)

### ✅ Выполнено

1. **Удалены кнопки навигации**
   - Убраны кнопки "Правила" и "Отслеживание" из всех страниц
   - Оставлены только: "Игроки", "Проверка", "Discord"
   - Файлы: `anticheat.html`, `check.html`, `secret-menu.html`

2. **Исправлен баг с UMA банами**
   - Теперь проверяется поле `unbanned` при проверке банов
   - Если игрок купил разбан, он не будет показываться как забаненный
   - Исправлено в: `api/uma.js`, `js/ConfigCheckerUMA.js`
   - Логика: `expires > now && !unbanned` (бан активен только если не истек И не разбанен)

3. **Добавлено прощальное модальное окно**
   - Показывается один раз при первом посещении
   - Сохраняется в localStorage (`fear_farewell_modal_shown`)
   - Красивый дизайн с анимациями
   - Кнопка "Прощай" для закрытия
   - Файл: `js/FarewellModal.js`

4. **Добавлена пасхалка с музыкой**
   - При клике на иконку Fear Protection включается музыка "Дипинс - Этажи"
   - Повторный клик останавливает музыку
   - Анимация иконки при воспроизведении
   - Уведомления о статусе воспроизведения
   - Файл: `js/easter-egg.js`

5. **Защита от DevTools**
   - Заблокирован F12
   - Заблокирован Ctrl+Shift+I (Inspect)
   - Заблокирован Ctrl+Shift+J (Console)
   - Заблокирован Ctrl+Shift+C (Inspect Element)
   - Заблокирован Ctrl+U (View Source)
   - Отключено контекстное меню (правый клик)
   - Файл: `js/easter-egg.js`
